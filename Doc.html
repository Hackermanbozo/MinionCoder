Comment is a comment it does nothing to the code and only can be on a new line example "// Hello this is a comment" after the two slashes there has to be a space

IO lib is a libary needed to run the functions CreateFile,CreateFolder,Open,WriteText to import it just type "Import IO"

CreateFile is a function that creates a file example "CreateFile C:\Hello.txt" you just need to input the path

CreateFolder is the Same as CreateFile but creates a folder instead of a file example "CreateFolder C:\Hello"

WriteText is a fuction that Writes text to a file example "WriteText C:\Hello.txt Hello"

Open is a function that opens some thing either a website or file just need the path "Open https://www.google.com"
